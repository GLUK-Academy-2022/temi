# python flask api

clone repo

    git clone https://github.com/temialao/python-api.git

install python3

    brew install python3

Check it's installed

    python --version

check pip3 is installed

    python3 -m pip --version

intall flask

    pip3 install flask

run app

    python3 main.py

or

    flask --app main run

view page at:

    http://127.0.0.1:8000/hello
